Functions

Creating and Calling Function

  • Function is a part of code which we can use anywhere
  • We can pass some parameter and we can do certain operations using the parameter and return a result or print something
  • We can define a function using "def" in python
  • To call a function use the function name followed by parenthesis (empty if no parameter,else required parameters

Example

def my_function()
  print("Hello World")

my_function() #prints Hello World

Arguments

  • Information can be passed to the function as arguments
  • We can add as many arguments as we want, use the operations and can return the output
  • If we give the Incorrect number of arguments it will pop up an error

Example

myfunc(firstname,lastname)
  return firstname+" "+lastname

print(myfunc("Ram","Krishna"))
#Ram Krishna

Recursion

  • We can call the same function within the same function
  • But we should be careful with the end of the recursion

Example

myrecu(k)
  if (k>0):
    result=k+myrecu(k-1)
  else:
    result=0
  return result

print(myrecu(6)) #21
