Sets

  • Sets are used to store the multiple items in a single variables
  • set is collection of object which is unordered and unindexed
  • sets are written with curly brackets
  • unordered means there is no specific order,it can appear in different orders every time you use them
  • Sets cant have the two items with same values
  • Sets can have different data types

Example

set1={1,2,3,"python",True}
#can have any type
set2={1,2,3,1}
print(set2)#prints {1,2,3}
#sets cant have duplicate values

Add and Remove

  • To add one element we can use "add()"
  • To add items from one set to another set we can use the "update()"
  • if we use the + operator then we will get the union of 2 sets
  • We can remove a element from the set by using "remove()", place the object in brackets
  • if item is not present then remove() will raise an error
  • We can also use the "discard()" function which will not pop up any error when the element is not present in the set
  • We can use the "pop()" to remove the last item, but sets are unordered so we dont know which items that gets removed
  • return value of the pop() is the removed item
  • "clear()" will empty the set
  • del will delete the set completely

Example

set1={1,2,3}
set2={4,5}
set3=set1+set2 #set3 will be {1,2,3,4,5}
set1.add(4) #adds 4 to the set
set1.update({4,5,6})
print(set1) #prints {1,2,3,4,5,6}
set1.discard(6) #removes 6
x=set1.pop() print(x) #prints 1 print(set1) #prints {2,3,4,5} set1.clear() #remove all elements del set1 #deletes the set

Lists

  • lists are also used to store the multiple values
  • Lists are created using square brackets
  • List items are ordered with 0 indexed, changeable, and allow duplicate values
  • List items can be of any data type

    • Indexing

  • List items are indexed and you can access them by referring to the index number
  • Negative indexing means start from the end
  • You can specify a range of indexes by specifying where to start and where to end the range
  • When specifying a range, the return value will be a new list with the specified items
  • By leaving out the start value, the range will start at the first item
  • By leaving out the end value, the range will go on to the end of the list
  • Specify negative indexes if you want to start the search from the end of the list

Example

list1=[1,2,3,1]
list2=[True,"python",1,2,3]
print(list1[2]) #prints 3
print(list2[:-1]) #prints all elements except the last element
print(list2[1:]) #prints ['python', 1, 2, 3]
print(list2[1:3]) #prints ['python', 1]

Add and remove

  • To add an item to the end of the list, use the "append()"
  • To insert a list item at a specified index, use the "insert()"
  • To append elements from another list to the current list, use the "extend()"
  • By using + operator we can add one list on another list
  • The "remove()" method removes the specified item
  • The "pop()" method removes the specified index
  • If you do not specify the index, the "pop()" method removes the last item
  • The "del" keyword also removes the specified index
  • The "del" keyword can also delete the list completely
  • The "clear()"" method empties the list

Example

list1=[1,2]
list2=[True,"python",1,2,3]
list3=list1+list2
print(list3) #prints [1,2,True,"python",1,2,3]
list1.append(3) #adds 3
list1.insert(1,"python")
print(list1) #prints [1,"python",2]
list1.remove("python") #removes python
list2.pop(0) #removes True
del list2[0]
print(list2[0]) #prints 1
del list1 #deletes the list1
list2.clear() #list2 will be empty
