Python Variables

Variables

Variables are containers for storing data values.

Creating Variables

Python has no command for declaring a variable.
A variable is created the moment you first assign a value to it.
Variables do not need to be declared with any particular type, and can even change type after they have been set.

Example

x=5 #assigning value 5 to x
y="CS251" #y is of type string
y=251 # now y is of type int

Variable Names

There are some rules to name a variable in python

  • Variable names cant start we a number
  • Variable names can only contain the alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case sensitive (x and X both are different)

Variable name can have short name like x and X or a more descriptive way like age

Example

#valid names
name="CS251"
_name="CS251"
course_name="CS251"
Course_name="CS251"
#invalid names
251course="CS251"
course name="CS251"

Global Variables

Variables that are created outside of a function(we will see later) are known as global variables
Global variables can be used by everyone, both inside of functions and outside
If you create a variable with the same name inside a function, this variable will be local, and can only be used inside the function. The global variable with the same name will remain as it was, global and with the original value.
